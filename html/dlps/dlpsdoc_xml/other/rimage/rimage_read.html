Rimage Workflow for Batch-Reading Archival Discs

This document describes the workflow for reading a batch of archival discs using the Rimage system at Digital Library Production Services (DLPS), University of Virginia Library. It is intended for use by internal DLPS staff.

This workflow makes it possible to transfer large sets of data from CDs or DVDs to a hard disk (or any disk or server accessible to the Rimage PC) with minimal effort.

Note: This documentation pertains to version 6.5 of the Rimage OfficeNet software.

Set Up
  • Open the Rimage System Manager (Start → All Programs → Rimage → Rimage System Manager).
  • Check the current setting for media type:
    • Click the main Settings... button near the picture of the Rimage 2000i (or the picture of the Desktop 1500, as the case may be).
    • In the Bins/Media tab, under the Media: section, check the Media type setting. It will be set to either DVDR or CDR.
    • If the setting already corresponds to the type of discs you will be burning, then just click OK. If it doesn't, you must set it to the type of media you will be using:
      • Click Cancel to close the settings window.
      • Pause the Production Server by clicking its "pause" button (looks like a VCR pause button with the || symbol).
      • Click the main Settings... button near the picture of the Rimage 2000i (or the picture of the Desktop 1500, as the case may be).
      • In the Bins/Media tab, under the Media: section, change the Media type setting to DVDR or CDR as appropriate.
      • Click OK to save the settings and close the settings window.
      • Click the "resume server" button for the Production Server (looks like a VCR play button with a green triangle).
  • Pause the production server (if you haven't done so already in a previous step) by clicking the "pause" button (looks like a VCR pause button with the || symbol).
  • Click the Settings... button for the Production Server.
  • If the Server and Reading tabs are not visible, click the More>> button.
  • Set the following options:

    Tab Option Setting to use Description
    Server Operating Mode Automatic Bulk Read This puts the Production Server in automated batch-reading mode.
    Server Use FIFO mode Checked, if disc order is relevant If maintaining the original sequence of the discs is important, check this checkbox to enable first-in-first-out (FIFO) mode, so that the discs are output in the same order by which they were input. If the original disc sequence is irrelevant, you can leave this checkbox cleared to increase the overall speed of the process.
    Reading Base Folder for Images Any valid path, such as C:\temp — though normally you will want to use an external drive with plenty of free disk space, typically E:\ This is the location to which folders and files from the disc will be copied.
    Reading Use Volume ID ... as Image Folder Name Checked For each disc, a new folder with the same name as the disc name will be created to contain the folders/files from that disc.
    Reading Extract ISO9660 Images to Folders and Files Checked As each disc is read, the data from that disc will be copied as actual folders and files, rather than as a single large disc image (.img) file containing the entire contents of the disc.
  • Click OK to save the settings and close the settings window.

Read Discs
  • Load the input bin with the DVDs or CDs to be read. (On the Desktop 1500, the input bin is the one with the three prongs to hold the discs. On the 2000i, the input bin is the left bin.)
    TIP

    If you have enabled FIFO (first-in-first-out) mode (see above for details), you can place the discs in the input bin in reverse order, and when reading is complete they will be in the desired order. For example, if you have three discs to read, put disc 1 on the bottom of the stack, disc 2 in the middle, and disc 3 on top. Disc 3 will be picked up and read first, and placed on the ouput spindle first. Disc2 will be output second, and disc 1 will be output last, so that it is on top of the output stack. That way, the discs are in the correct order at the end of the process.

  • To begin reading discs, click the "resume server" button (looks like a VCR play button with a green triangle) for the Production Server. The arm of the Rimage will place a disc in each of the two CD/DVD drives and begin reading the discs.
  • While the discs are being read, you can click the Server Orders tab to view the status of the current discs, and for any error messages.
  • When all discs in the input bin have been retrieved, the Rimage System Manager will display an error dialog saying that there are no more discs in the input bin. If there are more discs to be read, put them in the input bin and click Retry. If you're finished, click Stop Line.
  • When you're finished reading all discs, return the Production Server to its normal state:
    • Click the "pause" button (looks like a VCR pause button with the || symbol) for the Production Server.
    • Click the Server Settings... button.
    • On the Server tab, set the Operating Mode to Normal. Click OK.
    • Click the "resume server" button (looks like a VCR play button with a green triangle).
