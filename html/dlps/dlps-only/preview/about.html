About TEI Preview

TEI Preview provides a no-frills HTML view of an underlying TEI XML document. Page images display in a separate window, as do notes.

Known issues

Columns

Issue: The stylesheet currently does not support representation of columns.
Work-around: None; text is always displayed in a single column.

Small capitals

Issue: The rendering of small capitals (<hi rend="small-caps">) relies on CSS (font-variant: small-caps). Browsers don't always render this style correctly, especially at smaller font sizes.
Work-around: Representation of small-caps is supplemented by underlining.

Gothic typeface

Issue: Rendering a gothic typeface (<hi rend="gothic">) would require a gothic-style font widely available across browsers and platforms, and there is no such font.
Work-around: Gothic type is represented in bold italic.
