Setup for TEI Header Workflow

Digital Library Production Services, University of Virginia Library

Last modified: December 28, 2005

Overview: The DLPS workflow for creating full TEI headers requires certain software to be present on the user's local PC. This document details the requirements.

  • Perl — Have your LSP install Perl. The standard port of Perl for Windows is called ActivePerl and is available as a free download from
    http://www.activestate.com/Products/ActivePerl/
    To test the installation, open a command window (Start → Programs → Accessories → Command Prompt) and enter: perl -v
  • Java — Have your LSP install the Java Runtime Environment, available as a free download from
    http://java.com/
    (Just click the Get It Now button.) To test the installation, open a command window (Start → Programs → Accessories → Command Prompt) and enter: java -version

  • NoteTab — Normally the free text editor NoteTab Light will already be installed on your PC, but if you have a version older than 4.8, have your LSP upgrade to the current version. NoteTab Light is available either from ITC's Software Central or from the NoteTab web site:
    http://www.itc.virginia.edu/desktop/central/
    http://www.notetab.com/
  • NoteTab setup
    • If you don't already have a folder named C:\temp on your computer, create it. (This folder is needed for temporary files and error logs.)
    • Make sure you have write permission on folder C:\temp and on whatever folder you use for the output file.
    • Download the file DLPS.clb (right-click and choose Save Target As..., then for file type, select "All files") and save it to C:\Program Files\NoteTab Light\Libraries
    • Close NoteTab if it's open. Open NoteTab, open the Clipbook (Tools → Clipbook → Open Clipbook), and choose DLPS from the pull-down menu in the Clipbook.
    • Double-click About these commands... to view a dialog box with a description of each command in the DLPS clip library.
    • Map the DLPS workspace \\pogo.lib.virginia.edu\text as a network drive, using any available drive letter.
    • Double-click Indicate drive letter for DLPS workspace... in the Clipbook. In the dialog box that appears, enter the drive letter used to map the DLPS workspace as a network drive.
