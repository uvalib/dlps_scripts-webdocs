Setup for TEI Header Workflow

Digital Library Production Services, University of Virginia Library

Last modified: November 2, 2006

Overview: The DLPS workflow for creating full TEI headers requires certain software to be present on the user's local PC. This document details the requirements.

  • Perl — Have your LSP install Perl. The standard port of Perl for Windows is called ActivePerl and is available as a free download from
    http://www.activestate.com/Products/ActivePerl/
    To test the installation, open a command window (Start → Programs → Accessories → Command Prompt) and enter: perl -v
  • Java — Have your LSP install the Java Runtime Environment, available as a free download from
    http://java.com/
    (Just click the Get It Now button.) To test the installation, open a command window (Start → Programs → Accessories → Command Prompt) and enter: java -version

  • Oxygen — The preferred XML editor is Oxygen. It's a commercial product (DLPS and Cataloging have licenses for the workstations where Oxygen is installed), but the latest version of Oxygen is available as a free download from
    http://www.oxygenxml.com/
  • Setup for the MARC-to-TEI conversion script
    • If you don't already have a folder named C:\temp on your computer, create it. (This folder is needed for temporary files.)
    • Make sure you have write permission on folder C:\temp and on whatever folder you use for the output file.
    • Map the DLPS workspace \\pogo.lib.virginia.edu\text as a network drive, using any available drive letter.
    • Download the file marc2tei.vbs (right-click and choose Save Target As..., then for file type, select "All files") and save it to any convenient location. You may want to create a folder such as C:\marc2tei for this purpose.
  • Oxygen setup (optional)
    • Open Oxygen; go to Tools → External Tools → Preferences
    • Click New to set up a new external tool
    • Under "Name", enter "MARC to TEI" (or other suitable name)
    • Under "Command line" enter "WScript.exe" followed by the full path to this script file. For example: WScript.exe C:\marc2tei\marc2tei.vbs
    • Click OK; click OK again to dismiss the Preferences window
    • The MARC-to-TEI conversion script is now available within Oxygen. Go to Tools → External Tools → MARC to TEI, or just click "MARC to TEI" in the main Oxygen toolbar.
    Note: When the MARC-to-TEI conversion is complete, the script attempts to open the TEI XML output file in your preferred XML editor. The default editor is Oxygen 7.2:
    C:\Program Files\Oxygen XML Editor 7.2\oxygen7.2.exe
    If this program doesn't exist on your computer, the script will alert you and give you the opportunity to find your preferred XML editor using a standard file-selection window. The script will save the location of your preferred editor for future use. If you upgrade to a later version of Oxygen, you will need to edit the script's configuration file (named marc2tei.ini and located in the same folder as the marc2tei.vbs script file). The last line of the file contains the path to the preferred XML editor; change it to point to the latest version of Oxygen.
